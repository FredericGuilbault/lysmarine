The documentation of lysmarine is made with the help of docify,
The code of it is hosted [ Here ](https://gitlab.com/lysmarine/lysmarine)

To build the documentation into a website :

===== Install Docsify =====
```
npm i docsify-cli -g
```



===== Init the repository =====
```
docsify init -l true -t pure ./
````



===== Run the server =====
```
docsify serve ./
```
